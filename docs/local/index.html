本地部署

提示

常规方式访问为 CDN 访问 tesseract.js，如果需要本地访问，则需要如下操作：

1、将静态文件复制至本地服务器上​

在node_modules/idcard.js/tesseract移动至本地静态文件夹内：如 /public/、/static/ 等

2、在代码中添加本地路径​

recognize(file, {
  workerPath: "/idcard.js/tesseract/worker.min.js",
  corePath: "/idcard.js/tesseract/tesseract-core.wasm.js",
});
